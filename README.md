# YEG祭2026 会場案内（本番用）

## GitHub Pagesへのアップロード
このフォルダの中にある `index.html`、`.nojekyll`、`vercel.json`、`README.md` をリポジトリ最上位へアップロードしてください。

## 番号位置の変更
1. 公開ページ左下の「番号位置を調整」を押します。
2. 「編集開始」を押し、赤い番号をドラッグします。
3. 「変更版HTMLを保存」を押すと、新しい `index.html` がダウンロードされます。
4. GitHub上の古い `index.html` を、ダウンロードした新しい `index.html` で上書きしてください。

「この端末に保存」は、同じブラウザでの確認用です。公開サイトへ永久反映する場合は、必ず「変更版HTMLを保存」→GitHubへ上書きしてください。

## 出展者情報・タイムテーブル
`index.html` 内の `exhibitor-data` と `timetable-data` のJSONを編集できます。
