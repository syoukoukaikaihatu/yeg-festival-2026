YEG祭2026 25店舗・新レイアウト対応版

公開方法:
  index.html / .nojekyll / vercel.json をGitHubリポジトリの最上位へアップロードしてください。

番号位置の調整（管理者のみ）:
  公開URLの末尾に ?edit=1 を付けて開きます。
  例: https://example.vercel.app/?edit=1
  赤い番号（1〜25）と「縁日コーナー」をドラッグし、「変更版HTMLを保存」を押します。
  ダウンロードされた index.html をGitHubへ上書きしてください。
  通常URLでは編集パネルは表示されず、来場者は位置を変更できません。

縁日コーナー:
  通常表示で緑色の「縁日コーナー」をタップすると、設置内容が表示されます。
  初期内容: スーパーボールすくい、型抜き、輪投げ、水風船、めだかすくい。
